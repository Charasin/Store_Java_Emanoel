# Система за управление на магазин (Java)

Проектът реализира система за управление на магазин според университетското задание.

## Какво прави

- **Стоки** с ID, име, доставна цена, категория (хранителна/нехранителна), срок на годност и количество.
- **Продажна цена** = доставна цена + надценка (храна +20%, нехрана +30%).
- **Намаление** с 15% при наближаващ срок (под 3 дни).
- Стоки с **изтекъл срок** не могат да се продават.
- **Изключение** при недостатъчно количество (с информация колко не достига).
- **Касиери и каси** — на една каса работи един касиер.
- Продажба само при **достатъчно наличност и достатъчно пари** от клиента.
- **Касови бележки** с пореден номер, касиер, дата/час, списък със стоки и обща сума.
- Всяка бележка се записва в отделен `.txt` файл (с номера в името) и може да се чете обратно.
- **Сериализация и десериализация** на бележките (`.ser` файлове).
- **Магазин**, който смята разходи (заплати + доставки), приходи (оборот) и печалба.
- **JUnit тестове** за основните правила.

## Структура

```
src/main/java/
  Main.java          — демонстрация
  model/             — данните (Product, Cashier, CashRegister, Receipt, ReceiptItem, SoldProducts, ProductCategory)
  service/Store.java — бизнес логика (продажби и финанси)
  exception/         — собствени изключения
src/test/java/       — тестове (ProductTest, StoreTest)
```

## Изисквания

- JDK 17 или по-нова (разработено с Eclipse Temurin 26)
- Maven
- IntelliJ IDEA (по избор)

## Стартиране

**През IntelliJ:** File → Open → посочи папката → отвори `Main.java` → ▶ Run.
За тестовете: десен бутон на `src/test/java` → Run.

**През терминал:**
```bash
mvn test
mvn package
java -jar target/javastore-1.0.0.jar
```

След стартиране се създава папка `receipts/` с бележките (`.txt` и `.ser`).
